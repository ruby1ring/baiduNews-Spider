### 本项目是一个基于Golang的Colly框架搭建的百度新闻爬取、分析学习项目，可以实时查看新闻标题的情感分析.


## 🚩 **New Features/Updates**

- ✅ 百度新闻首页不同类别新闻的爬取方法

## 🔧 Dependencies and Installation

```##
#Dependencies
Golong 1.18
Colly  v2

#installation

go mod init spider
go mod tidt

```
